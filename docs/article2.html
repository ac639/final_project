How is the Web built?

As mentioned on the previous page, the web is built using different markup and programming languages and frameworks.
We will cover a few.

HTML

HTML stands for Hypertext Markup Language and is used for website document structuring.
With HTML, websites can contain text, links, images, and video.
The order in which these mediums appear can be controlled individually.
HTML uses tags to specify different types of texts, fonts, and colors.

CSS

CSS stands for Cascading Style Sheets and is a style sheet language to manipulate how HTML code is displayed.
CSS controls website presentation and how HTML elements appear on screen. CSS changes site layout, such as changing how lists, menus, tables, images or boxes of text appear on screen.

Javascript

Unlike HTML and CSS, Javascript is a programming language.
Javascript allows the ability for actions to occur from user input on a website. Javascript makes websites more interactive and allow for things on the page to move when you click on them, such as drop down menus, or form submitting when clicking a button.

Bootstrap

Bootstrap is a framework for quickly deploying websites. It is not a markup or programming language but rather a collection of HTML and CSS code wrapped in modular packages that can be mixed and matched to build and add funcitonality to websites. Rather than develop HTML and CSS separately and from scratch, you can use bootstrap "packages" that have all of this already built.
This entire website was made with Bootstrap.


It should be noted that HTML, CSS, Javascript, and Bootstrap are what as known as "open-source". Open-source allows anyone to copy, modify, and redistribute the code that creates these tools. This has led to these tools to become the standard for website developing. They are an open standard. This is incredibly important as it unifies the world wide web and ensures efficient use of resources where everyone is using common tools.
